# Grill Me

A skill that interviews you relentlessly about a plan or design until you reach a shared understanding — walking down each branch of the decision tree, resolving dependencies between decisions one at a time.

## Features

- **Relentless interviewing** — questions every aspect of your plan until nothing is left ambiguous
- **Decision-tree traversal** — walks each branch in order, resolving dependencies between decisions one-by-one
- **One question at a time** — keeps the conversation focused instead of dumping a wall of questions
- **Recommended answers** — for every question, it offers its own recommended answer to react to
- **Codebase-aware** — when a question can be answered by exploring the code, it explores instead of asking

## Installation

Users of this marketplace can install via:

```
/plugin install grill-me@ai-marketplace
```

Or manually copy `skills/grill-me/` to `~/.claude/skills/grill-me/`.

## Usage

The skill activates when you want to stress-test a plan, get grilled on your design, or say "grill me".
